# Atomic Test Plan — T1053.005 Scheduled Task/Job

**Target rule**: `100104` — "Scheduled Task created"
**Data source**: Windows Security EventID 4698
**Endpoint**: `win-ep-01`
**MITRE**: https://attack.mitre.org/techniques/T1053/005/

**Prereq**: Windows Security channel phải log "Task Scheduler" success. Verify:
```powershell
auditpol /get /subcategory:"Other Object Access Events"
```
Nếu "No Auditing" → bật:
```powershell
auditpol /set /subcategory:"Other Object Access Events" /success:enable
```

---

## Test 1 — schtasks /create

```powershell
Invoke-AtomicTest T1053.005 -TestNumbers 1
```

Chạy:
```
schtasks /create /F /tn "AtomicTask" /tr "cmd.exe /c echo atomic" /sc onlogon /ru System
```

**Expected**:
- Windows Security EID **4698** ("A scheduled task was created") với `taskName=\AtomicTask`
- Rule `100104` fires level 10

**Verify**:
```
rule.id : "100104" AND agent.name : "win-ep-01"
```

**Cleanup**:
```powershell
Invoke-AtomicTest T1053.005 -TestNumbers 1 -Cleanup
```
