# Atomic Test Plan — T1087.001 Local Account Discovery

**Target rules**:
- `100105` (Windows) — Sysmon EID 1, image = net.exe/whoami.exe/…
- `100115` (Linux) — auditd execve of whoami/id/getent

**MITRE**: https://attack.mitre.org/techniques/T1087/001/

---

## Windows — Test 1

```powershell
Invoke-AtomicTest T1087.001 -TestNumbers 1
```

Chạy:
```
net user
net localgroup administrators
net accounts
```

**Expected**:
- Sysmon EID 1 với image `net.exe`, commandLine chứa `user` / `localgroup`
- Rule `100105` fires level 5 (x3 alerts, mỗi command 1 alert)

**Verify**:
```
rule.id : "100105" AND agent.name : "win-ep-01"
```

**Cleanup**: no-op (test read-only).

---

## Linux — chạy tay (không có runner ART cho lab)

**Trên linux-ep-01**:
```bash
whoami
id
getent passwd
```

**Expected**:

- Rule `100115` fires level 5

**Verify**:
```
rule.id : "100115" AND agent.name : "linux-ep-01"
```

**Cleanup**: no-op.
