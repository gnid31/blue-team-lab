# Atomic Test Plan — T1059.001 PowerShell

**Target rule**: `100101` (custom) — "Suspicious PowerShell cmdline"
**Data source**: Sysmon EventID 1 (Process Create)
**Endpoint**: `win-ep-01`
**MITRE**: https://attack.mitre.org/techniques/T1059/001/

---

## Test 1 — PowerShell -EncodedCommand

Chạy PowerShell với `-EncodedCommand` (base64) — mô phỏng loader Cobalt Strike / Empire.

**Command (PowerShell as Admin trong VM Windows)**:
```powershell
Import-Module Invoke-AtomicRedTeam
Invoke-AtomicTest T1059.001 -TestNumbers 1
```

Test này chạy:
```powershell
powershell.exe -EncodedCommand JAB3AGMAPQBOAGUAdwAtAE8AYgBqAGUA…
```
(chuỗi base64 decode ra `Write-Host "Hello from the Atomic Red Team"`)

**Expected detection**:
- Sysmon EID 1: `image=powershell.exe`, `commandLine` chứa `-EncodedCommand`
- Rule `100101` fires level 12

**Verify**:
```
rule.id : "100101" AND agent.name : "win-ep-01"
```

**Cleanup**:
```powershell
Invoke-AtomicTest T1059.001 -TestNumbers 1 -Cleanup
```
(test này không tạo artefact, cleanup là no-op)
