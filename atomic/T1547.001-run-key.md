# Atomic Test Plan — T1547.001 Registry Run Key persistence

**Target rule**: `100108` — "Autorun key modified"
**Data source**: Sysmon EventID 13 (RegistryEvent SetValue)
**Endpoint**: `win-ep-01`
**MITRE**: https://attack.mitre.org/techniques/T1547/001/

---

## Test 1 — HKCU Run key

Ghi vào `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` — chạy khi user login.

**Command (PowerShell as Admin)**:
```powershell
Invoke-AtomicTest T1547.001 -TestNumbers 1
```

Test này chạy:
```
reg add "HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Run" /v Atomic /t REG_SZ /d "C:\Path\to\atomic.exe" /f
```

**Expected detection**:
- Sysmon EID 13: `targetObject=HKU\...\CurrentVersion\Run\Atomic`
- Rule `100108` fires level 12

**Verify**:
```
rule.id : "100108" AND agent.name : "win-ep-01"
```

**Cleanup**:
```powershell
Invoke-AtomicTest T1547.001 -TestNumbers 1 -Cleanup
```
(Cleanup `reg delete HKCU\...\Run /v Atomic`)
